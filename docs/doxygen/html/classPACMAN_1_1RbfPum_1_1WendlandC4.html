Detailed Description

A functor which computes a RBF function: WendlandC4 Each RBF function must have a host device method fp_t Eval(const / fp_t) const which computes an image of the RBF function And a method SetRadiusInv which is a setter for the function support radius.

Note
In this file, Wendland functions are computed with a no-branching strategy to avoid threads divergence and increase performance on device
